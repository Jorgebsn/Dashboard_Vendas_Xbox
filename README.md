📊 Dashboard de Vendas Xbox | Análise de Dados com Excel

Projeto desenvolvido durante o bootcamp da DIO com foco em transformar dados em insights estratégicos. Este dashboard interativo apresenta uma visão clara e objetiva das vendas de produtos Xbox, permitindo acompanhar indicadores de desempenho e apoiar a tomada de decisão.

Ao longo do projeto, foram aplicados conceitos de análise de dados, tratamento e organização de informações, além da criação de visualizações dinâmicas e intuitivas no Excel.

🔹 Análise de performance de vendas
🔹 Visualização de dados com gráficos dinâmicos
🔹 Organização e tratamento de base de dados
🔹 Foco em geração de insights para o negócio
